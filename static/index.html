上市公司套保公告与年报披露监控

这个 HTML 不连接真实后端，使用内置样例和浏览器本地存储，模拟“公告导入 → 关键词召回 → 结构化抽取 → 证据链 → 提示标签 → 前端展示”的最小闭环。后续真实项目可把这里的页面逻辑迁移到 React + FastAPI。

1. 公告导入内置样例 / 手动新增 / JSON 导入
2. 关键词召回标题 + 正文命中套保、衍生品、远期结售汇等词
3. 字段抽取用正则和规则模拟 LLM 抽取核心字段
4. 模型校验金额、币种、审批层级、证据句校验
5. 证据链字段值关联原文句子和模拟页码
6. 页面提示大额、股东会、亏损、低置信度等提示
