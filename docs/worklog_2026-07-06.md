## 七、已知坑 & 注意事项

1. **companies 表数据缺口**：
   - 企业性质（ent_type）暂无数据，留空
   - 市值（market_cap）因东方财富接口代理问题未获取
   - 如需补全，可用 Wind/Choice 导 CSV，或等 akshare 其他接口

2. **抽取积压**：
   - 2025 年回填的 4668 条 announcements 没有对应的 hedge_events
   - 日常自动抽取只碰"近几天"，历史积压必须手动触发 extract workflow
   - 建议分批抽，避免 LLM API 费用和超时

3. **巨潮反爬**：
   - Actions 海外 IP 敏感，回填时分年跑、季度间 sleep 1.5-2 秒
   - 被封了换本地跑 `scripts/backfill_announcements.py`

4. **额度口径陷阱**：
   - 商品套保金额 = 保证金最高占用额（千万级）
   - 外汇套保金额 = 业务总额（千万~百亿级）
   - 严禁跨类别直接求和或比较
