## 四、已知坑 & 注意事项

1. **巨潮反爬**：Actions 海外 IP 敏感，回填时分年跑、季度间 sleep 1.5-2 秒。真被封了换本地跑同一个脚本。
2. **额度口径陷阱**：商品套保金额是「保证金最高占用额」，外汇是「业务总额」，严禁跨类别直接求和或比较。
3. **抽取管线**：回填进来的历史公告，`extracted_at` 为空，日常自动抽取只碰"近几天"，必须用 workflow_dispatch 手动批量抽历史积压。
4. **cron 时区**：`daily-cninfo.yml` 现在是 UTC 19:00 = 北京次日 03:00，一次抓全前一整天。
5. **公司维表数据**：没有现成免费 API 能完美给出"企业性质"（央企/地方国企/民企），Wind/Choice 导 CSV 是最快路径。
