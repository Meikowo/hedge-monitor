## 4. 已发现问题与决策

- 全市场年度分类受 300 页上限截断，不能用于样本发现；已改定向查询。
- Windows 临时文件独占锁导致 PyMuPDF 无法打开；已改为内存字节打开，Linux/Windows 统一。
- LLM 会在数组中输出字符串 `None`；规范化层已过滤 nullish 字符串。
- 复杂表格的模型摘录常是重新拼接的“行名+列名+数值”，并非 PDF 展平文本的连续原句，
  因此 quote_verified=false 是正确的保守结果。未解决前不批量扩张、不进可信统计。
